## URl Shortening and Scrapper

This is a simple url shortening application developed using laravel 8

 ##Libraries used 
- For Scrapping  [laravel-goutte](https://github.com/dweidner/laravel-goutte) is used.
- For url shortening  [short-url](https://github.com/ash-jc-allen/short-url) is used.
